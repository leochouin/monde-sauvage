GASPESIE RIVER MODAL GUIDE
Purpose: numbering guide + short modal copy for a coding agent.
Source basis: handwritten Monde Sauvage map + real Gaspésie river references.
Important: some labels on the hand-drawn map are crystal clear; others were inferred from geography because the handwriting is stylized.
Recommendation: keep the IDs stable, but visually recheck any entry marked confidence=medium before hard-coding final production copy.

NUMBERING LOGIC
- IDs are assigned in a practical map-reading order from west/left toward east/right, then to the lower-right river cluster.
- Use the "map_position_hint" to know which river is which when looking at the illustration.
- If your SVG/path layer already has separate river paths, map each path to the matching river_id below.

ALIASES / NORMALIZATION NOTES
- "Pabos" on the map may refer to the 3-river Pabos system in general, not a single standalone river card.
- "du Grand Pabos" => normalize to "Grand Pabos".
- "du Grand Pabos Ouest" => normalize to "Grand Pabos Ouest".
- Some north-coast names were inferred from location and regional geography because the handwriting is harder to read there.

IMPLEMENTATION NOTES FOR CODING AGENT
1. Use river_id as the stable key.
2. Use name for the card title.
3. Use modal_blurb as the short body text.
4. Use map_position_hint during manual QA when matching SVG paths to data entries.
5. For entries with confidence=medium, confirm the handwritten label visually before shipping.
6. If you only want the strongest, safest set for V1, prioritize: 1, 2, 4, 5, 9, 10, 11, 13, 14, 15, 16.
7. If image_url is empty, fall back to a local placeholder or a later curated asset.
8. Wikimedia hotlinks may redirect; for production, consider downloading and self-hosting the chosen image.
